# Cline Agent Chess Game Tests

This directory contains test scripts for testing the Cline agent's ability to create a chess game through the API. These tests focus on the end functionality rather than code comparison.

## Test Scripts

### 1. test-chess-api.sh

This script tests the API interaction with the Docker Cline agent server, focusing on:

- Testing if the API server is running
- Testing API authentication
- Testing a hypothetical chess game creation endpoint
- Using the direct Anthropic API as a fallback
- Generating a chess game script using Claude 3.7

To run this test:

```bash
./test-chess-api.sh
```

### 2. test-cline-chess.sh

This script focuses specifically on using the Cline agent to create a chess game, with an emphasis on the end functionality:

- Checking if the API server is running
- Sending a task to create a chess game to the Cline agent
- Using the direct Anthropic API to demonstrate the expected functionality
- Testing the generated chess game script
- Explaining what would be needed for a full implementation

To run this test:

```bash
./test-cline-chess.sh
```

### 3. chess_game.sh

This is a simple bash chess game that demonstrates how the Cline agent could be used to create a chess game. It includes:

- A visual chess board in the terminal
- Support for making moves using algebraic notation
- Integration with the Cline agent API for move suggestions
- Fallback to direct Anthropic API if the Cline agent API is not available

To run the chess game:

```bash
./chess_game.sh
```

## Prerequisites

- Docker container with the Cline agent running (for API tests)
- Bash shell
- curl command-line tool
- Internet connection (for Anthropic API fallback)
- Anthropic API key (stored in memory-bank/api_keys.md)

## Running the Tests

1. Start the Docker container with the Cline agent:

```bash
./run-docker.sh --build --run
```

2. Run the test scripts:

```bash
./test-chess-api.sh
./test-cline-chess.sh
```

3. Play the chess game:

```bash
./chess_game.sh
```

## Test Results

The test scripts will generate the following files:

- `anthropic_chess_game.sh`: A chess game script generated by the Anthropic API (from test-chess-api.sh)
- `cline_chess_game.sh`: A chess game script generated by the Anthropic API (from test-cline-chess.sh)

These files demonstrate what would be possible with a full Cline agent API implementation.

## Notes

- The current API server only supports the `/api/state` endpoint
- The tests simulate what would happen if we had a more complete API
- The direct Anthropic API is used as a fallback to demonstrate the expected functionality
- In a full implementation, the Cline agent would generate the chess game script within the Docker container

## Future Enhancements

To fully implement chess game creation using the Cline agent API, the following would be needed:

1. Extended API Server:
   - Add endpoints for task submission and retrieval
   - Implement WebSocket support for real-time updates
   - Add file system access for saving and loading games

2. Cline Agent Integration:
   - Configure the Cline agent to handle chess-specific tasks
   - Set up the agent to generate and execute bash scripts
   - Implement a way to return the generated script to the user

3. Docker Container Enhancements:
   - Ensure the container has all dependencies for chess game development
   - Configure proper permissions for script execution
   - Set up persistent storage for saved games

4. Testing Framework:
   - Create tests for the chess game functionality
   - Verify move validation and game rules
