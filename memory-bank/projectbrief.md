# Project Brief: Cline Agent

## Overview
This project is a fork of the VSCode extension Cline, which is an autonomous coding agent. The primary goal is to create an API layer that allows the Cline extension to be integrated with a larger web platform.

## Core Objectives
1. Create an API server that communicates with the Cline extension
2. Containerize the VSCode extension with Docker for isolated execution
3. Develop testing frameworks for the API server

## Current Status
1. ✅ The frontend has been moved to a separate private project (cline-frontend-private)
2. The API server and Docker container setup are maintained in this fork
